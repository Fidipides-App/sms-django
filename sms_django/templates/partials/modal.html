Pesquisa

Para pesquisar, basta usar a tecla Enter do teclado, após inserir o número de telemóvel a pesquisar.

Os dados apresentados podem ser importados através de ficheiros CSV ou SQL.

Campo Tipo
id Númerico, único.
nome Texto
email Texto
localidade Texto
telefone Texto, único.
observacoes Texto
data_nascimento Texto, formato DD/MM/YYYY.

SMS

O módulo de SMS permite o envio de mensagem SMS individuais ou em massa, mediante o upload de um ficheiro CSV, com o formato abaixo descrito.

Campo Tipo
id Númerico, único.
telefone Texto, com indicativo.
mensagem Texto

Exportar

O módulo Exportar permite a exportação dos registos da base de dados para um ficheiro CSV, com o formato acima descrito.
